    AstronomicalTime.Timescale — Type.

    All timescales are subtypes of the abstract type Timescale. The following timescales are defined:

    • UTC — Coordinated Universal Time

    • UT1 — Universal Time

    • TAI — International Atomic Time

    • TT — Terrestrial Time

    • TCG — Geocentric Coordinate Time

    • TCB — Barycentric Coordinate Time

    • TDB — Barycentric Dynamical Time

    source
    AstronomicalTime.@timescale — Macro.
    @timescale scale

    Define a new timescale and the corresponding Epoch type alias.

    Example

    julia> @timescale Custom

julia> Custom <: Timescale
true
julia> CustomEpoch == Epoch{Custom}
true
